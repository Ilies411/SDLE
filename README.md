## Large Scale Distributed Systems (SDLE) - FEUP

### Instructions for running the application:

### Windows:

#### Execute the following programs:
Run server:

    runserver.bat

Run client:

    runclient.bat

Run loadbalancer:

    runloadbalancer.bat

Run server broker:
    runbroker.bat

Run a new server to add it into the ring then (not mandatory just for demo):
    run5560.bat
### MacOS:

#### Execute the following programs:
Run server:

    runserver.sh

Run client:

    runclient.sh

Run loadbalancer:

    runloadbalancer.sh

Run server broker:
    runbroker.sh

Run a new server to add it into the ring then (not mandatory just for demo):
    run5560.sh
