## Large Scale Distributed Systems (SDLE) - FEUP

### Instructions for running the application:

### Windows:

#### Execute the following programs:
Run server:

    runserver.bat

Run client:

    runclient.bat

Run loadbalancer:

    runloadbalancer.bat

### MacOS:

#### Execute the following programs:
Run server:

    runserver.sh

Run client:

    runclient.sh

Run loadbalancer:

    runloadbalancer.sh
